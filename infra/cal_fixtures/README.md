# Cal.com Fixtures

This directory is intended to store fixture files for testing the local Cal.com instance.

## Purpose

You can place mock data here, such as:

- JSON files for pre-filling user profiles.
- Configuration snippets for testing specific event types.
- Scripts for automating the setup of your local Cal.com instance after a fresh install.

This helps ensure that anyone setting up the project can quickly get a standardised testing environment for the scheduling component of the Conversion Engine.
